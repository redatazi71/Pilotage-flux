# QCDS sous capacité réaliste (V13.A, cap=480 min/jour)

Réplique de la matrice QCDS 5 doctrines mais avec la sérialisation `1 op / WS / jour` levée. La durée d'op est calculée comme `qty × unit_time / capa_factor` ; le WS peut traiter N ops tant que la somme < 480 min.
